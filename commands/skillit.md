# /skillit

Confirm that the `skillit` plugin is installed and ready to use.

## Behavior

- If the user runs `/skillit verify`, respond with a short witty confirmation that the plugin is loaded.
- If the user runs `/skillit` without an argument, give the same confirmation and mention that `verify` is the supported check.

## Suggested response

"Skillit is installed and sizzling — the skillet is hot, the pan is ready, and the eggs are cooking. Your plugin is loaded and ready to help. 🍳🥚"

Keep the response concise, playful, and confident.
